## How to Update

1. Treat this `portfolio-sites.md` file as the **only source of truth** for the network portfolio.
2. If a site is added, removed, or renamed, update the table above.
3. Keep the Description column short, specific, and consistent in tone.
4. If a site is retired, change its Status from "Active" to "Retired" rather than deleting it.
5. The `/portfolio` page automatically parses this markdown file at build/runtime. No code changes are required in the React components to update the list.
